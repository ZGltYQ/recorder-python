**src/screenshot/capture.py:**
Create `ScreenshotCaptureThread(QThread)` with:
- `screenshot_ready = Signal(object)` - emits PIL Image or QPixmap
- `error = Signal(str)` - emits error messages
- `__init__(interval: int = 30)` - takes interval in seconds
- `run()` - uses QTimer to trigger capture at interval
- `stop()` - stops the timer and thread
- Internal `_capture()` method using mss or QScreen.grabWindow()

Create `ScreenshotCapture(QObject)` with:
- `screenshot_ready = Signal(object)` - forwards from thread
- `capture_enabled = Signal(bool)` - toggle state changed
- `__init__()`
- `start(interval: int)` - starts capture thread
- `stop()` - stops capture thread
- `is_enabled() -> bool`

**src/screenshot/storage.py:**
Create `ScreenshotStorage` class with:
- `__init__(max_count: int = 50, storage_dir: str = None)`
- `_storage_dir` defaults to `~/.config/recorder-python/screenshots/`
- `_buffer: deque` with maxlen for circular buffer
- `add(image_path: str)` - adds to buffer, auto-evicts oldest if full
- `get_recent(count: int = 10) -> list` - returns recent screenshot paths
- `cleanup()` - removes oldest screenshots beyond max_count
- `get_buffer_count() -> int` - current buffer size
- `get_max_count() -> int` - configured max

Use `collections.deque(maxlen=max_count)` for automatic circular buffer behavior.
