# AI Analysis Prompt (from CONTEXT.md)

**Task Detection:**
```
You are analyzing a screenshot for actionable tasks.
Identify any: TODOs, questions asked, action items, decisions made, 
or items requiring follow-up.

Return a JSON list with this format:
[
  {
    "task": "description of the task",
    "priority": "high|medium|low",
    "context": "relevant context from screenshot"
  }
]

If no tasks found, return: []
```

**Task Auto-Solve:**
```
A screenshot analysis found this actionable task:
{task_description}

Based on the screenshot context:
{screenshot_context}

Provide a solution or recommendation for completing this task.
Be specific and actionable.
```

```python
def _on_screenshot_tasks_found(self, tasks: list):
    """Display screenshot tasks in side panel."""
    for task_info in tasks:
        task = task_info.get('task', '')
        solution = task_info.get('solution', '')
        priority = task_info.get('priority', 'medium')
        
        # Format with priority indicator
        priority_emoji = {'high': '🔴', 'medium': '🟡', 'low': '🟢'}.get(priority, '⚪')
        display_text = f"{priority_emoji} Screenshot Task\n\n{task}\n\n📋 Solution:\n{solution}"
        
        self.suggestions_widget.add_suggestion(
            f"Screenshot: {task[:50]}...",
            display_text
        )
```
